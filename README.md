# File Encryption and Decryption in C

This is a simple C mini project that demonstrates how file encryption and decryption works using basic character manipulation.

The project is useful for beginners who want to understand file handling and basic security concepts in C programming.

---

## Features
- Encrypt a text file using a simple logic
- Decrypt the encrypted file back to original form
- Uses file handling in C
- Beginner-friendly mini project
